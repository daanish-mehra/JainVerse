# GitHub Repository Setup Instructions

## Create Private Repository on GitHub

1. Go to https://github.com/new
2. Repository name: **JainVerse**
3. Description: "Where Ancient Wisdom Meets Modern AI - Jain Hackathon 2025"
4. Visibility: **Private** ✅
5. Click "Create repository"

## Push Code to GitHub

After creating the repository, run these commands:

```bash
cd /Users/daanish/Desktop/JainAI/jainverse

# Add remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/JainVerse.git

# Push to GitHub
git branch -M main
git push -u origin main
```

## Verify Push

1. Go to your GitHub repository: `https://github.com/YOUR_USERNAME/JainVerse`
2. You should see all commits and files
3. Verify all 5 commits are visible
